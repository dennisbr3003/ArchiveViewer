You can create new project using java language in Android Studio Meerkat version using below steps:

1. Open Android Studio
2. click on new project
3. Select Empty Views Activity
4. Click on next button
5. Then select Java from Language drop down section
6. Then select Groovy DSL (build.gradle) from Build Configuration Language drop down
7. Now click Finish button.

HandBrake Video Compression Settings for Android:

1. **Preset Selection:**
   - Use a device preset: "Android 720p30" or "Android 480p30" for compatibility.

2. **Video Settings:**
   - Codec: H.264 (x264)
   - Framerate (FPS): 48
   - Framerate Setting: Constant Framerate
   - Quality: Constant Quality RF slider between 20-23 (25)

3. **Audio Settings:**
   - Codec: AAC
   - Bitrate: 128 kbps

4. **Dimensions:**
   - Resolution: Adjust to match device screen size (720p or 480p recommended)

5. **Filters:**
   - Use defaults unless specific adjustments are necessary.

6. **Advanced Settings (optional):**
   - Adjust only if further file size reduction is needed; be cautious as it may impact compatibility.

7. **Process:**
   - Save destination: Set your destination folder
   - Start Encoding: Click “Start Encode” to compress the video

8. **Verification:**
   - Test the output video on your Android device to ensure quality and compatibility.
